# FORM/01

An original, full-screen editorial product experience for a fictional premium headphone. The site combines a Swiss-inspired monochrome grid with custom-generated campaign assets, scroll-scrubbed product storytelling, and a real-time WebGL acoustic field.

## Run locally

```bash
npm install
npm run dev
```

Production build:

```bash
npm run build
npm run preview
```

## Interaction map

- Lenis smooth scrolling synchronized with GSAP ScrollTrigger.
- Motion-powered hero word cycle plus GSAP parallax.
- A scroll-scrubbed canvas sequence that opens the assembled headset into its exploded components.
- A lazy-loaded Cobe WebGL listening field with drag and keyboard orbit controls.
- Working Focus/Aware and Field Active/Paused controls.
- Scroll-linked endurance counter and final purchase transition.
- Functional reservation drawer with finish selection and confirmation state.
- Responsive layouts and a reduced-motion path.

## Original assets

All product renders and editorial photography in `public/assets` were generated specifically for FORM/01. The headphone design, name, interface, copy, and campaign direction are original to this project.
